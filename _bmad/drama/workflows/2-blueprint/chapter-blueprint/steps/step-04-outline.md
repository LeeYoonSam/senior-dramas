# Step 4: 챕터별 아웃라인

## 목표
8챕터 각각의 상세 아웃라인을 작성합니다.

## 챕터 아웃라인 구조

각 챕터에 대해:

### 1. 챕터 제목
감정을 자극하는 은유적 제목

### 2. 핵심 장면 (3-5개)
- 장면 위치 (도입/중반/끝)
- 장면 요약
- 등장인물
- 핵심 대사 (1-2개)

### 3. 감정 곡선
- 챕터 시작 감정
- 전환점
- 챕터 끝 감정

### 4. 후킹 요소
- 챕터 끝 클리프행어
- 다음 챕터로 이끄는 질문

## 수행 작업

### 1. 챕터별 아웃라인 생성

**챕터 1 예시:**
```yaml
chapter_1:
  title: "평온한 아침의 그림자"

  key_scenes:
    - scene_1:
        location: "도입"
        description: "평화로운 아침 식사 준비, 하지만 시어머니의 한마디에 굳어지는 표정"
        characters: ["주인공", "시어머니"]
        key_dialogue: "밥맛이 예전 같지가 않네."

    - scene_2:
        location: "중반"
        description: "혼자 설거지하며 한숨, 30년 전 신혼 시절 회상"
        characters: ["주인공"]
        key_dialogue: (내적 독백) "그때는 정말... 행복할 줄 알았는데."

    - scene_3:
        location: "끝"
        description: "며느리와의 미묘한 갈등 시작, 손주 양육 방식 차이"
        characters: ["주인공", "며느리"]
        key_dialogue: "요즘은 다 그렇게 해요, 어머니."

  emotional_arc:
    start: "일상적 피로"
    turning_point: "무시받는다는 느낌"
    end: "불안한 예감"

  chapter_hook: "그날 밤, 시어머니 방에서 들려온 전화 통화... 누구와 무슨 이야기를?"
```

### 2. 챕터 흐름 검토

8챕터 전체 흐름을 요약해서 확인:

> **8챕터 흐름 요약**
>
> Ch1: 평온 → 불안의 씨앗
> Ch2: 갈등 폭발 → 상처
> Ch3: 고립 → 결심
> Ch4: 조력자 등장 → 희망
> Ch5: 비밀 폭로 → 충격
> Ch6: 최악의 순간
> Ch7: 클라이맥스 → 깨달음
> Ch8: 화해 → 새로운 시작
>
> 이 흐름이 괜찮으신가요?

### 3. 사용자 피드백 반영

수정 요청 시 해당 챕터 수정
