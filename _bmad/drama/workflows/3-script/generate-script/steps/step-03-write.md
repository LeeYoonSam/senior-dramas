# Step 3: 대본 작성

## 목표
선택한 챕터의 상세 대본을 작성합니다.

## 대본 작성 규칙

### 분량
- 기본: 3,000자 (한글 기준)
- 예외 (1, 7, 8장): 3,200-3,500자

### 3가지 서사 도구 활용

#### 1. 내레이션 (일상/감정 묘사)
- 주인공 시점의 상황 묘사
- 격식체 사용 (-습니다, -ㅂ니다)
- 감각적 디테일 포함

#### 2. 대사 (가족 간 대화)
- 서브텍스트 활용 (겉말과 속마음 다르게)
- 대사 블록 형식 사용
- 세대별 말투 특성 반영

#### 3. 내적 독백 (솔직한 속마음)
- 괄호 또는 이탤릭으로 표시
- 진심, 원망, 걱정 표출
- 여성 특유의 감정 표현

### 후킹 전략 적용

#### 오프닝 훅 (챕터 시작)
- 충격적 장면 회상 또는
- 감정적 대화로 시작

#### 마이크로 훅 (매 장면)
[상황 제시] → [갈등] → [감정 상처] 반복

#### 챕터 훅 (챕터 끝)
- 비밀/관계 변화 암시
- 다음 챕터 궁금증 유발

## 수행 작업

### 1. 대본 초안 생성

청사진의 챕터 정보를 바탕으로 대본 작성:

```
INPUT:
- chapter_outline: {chapter from blueprint}
- characters: {relevant characters}
- previous_summary: {if chapter > 1}
- style_guides: {loaded styles}

OUTPUT:
- script_content: 3000자 대본
- image_prompts: 3개 핵심 장면 프롬프트
```

### 2. 이미지 프롬프트 생성

각 챕터당 3개 핵심 장면:

```json
{
  "scene1": {
    "location": "챕터 도입부",
    "situation": "장면 상황 2-3줄",
    "characters": "등장인물 정보",
    "imagePrompt": "영문 이미지 프롬프트"
  },
  "scene2": {...},
  "scene3": {...}
}
```

### 3. 글자수 확인

- 목표 분량 확인
- 부족 시 디테일 추가
- 초과 시 불필요한 부분 정리
