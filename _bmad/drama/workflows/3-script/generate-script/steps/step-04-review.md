# Step 4: 검토 및 수정

## 목표
생성된 대본을 검토하고 필요시 수정합니다.

## 수행 작업

### 1. 대본 미리보기

생성된 대본을 사용자에게 제시:

```
┌─────────────────────────────────────────────┐
│ 📖 챕터 {N}: {title}                          │
├─────────────────────────────────────────────┤
│ 분량: {char_count}자 / 목표: {target}자       │
└─────────────────────────────────────────────┘

{script_preview - 처음 500자}

...

[전체 보기 / 계속]
```

### 2. 품질 체크리스트

> **품질 확인 항목:**
>
> ✅ 1인칭 주인공 시점 유지
> ✅ 3가지 서사 도구 (내레이션/대사/독백) 활용
> ✅ 오프닝 훅 적용
> ✅ 챕터 끝 클리프행어
> ✅ 목표 분량 달성
> ✅ 캐릭터 일관성
> ✅ 이전 챕터와 연속성

### 3. 수정 요청 처리

> 수정이 필요한 부분이 있으신가요?
>
> 1. **특정 장면 수정** - 장면 번호와 수정 방향
> 2. **대사 수정** - 특정 대사 변경
> 3. **감정 톤 조절** - 더 강하게/약하게
> 4. **분량 조절** - 늘리기/줄이기
> 5. **완료** - 현재 상태로 저장

### 4. 이미지 프롬프트 확인

> **이미지 프롬프트 확인:**
>
> **Scene 1** - {location}
> {situation_summary}
> ```
> {english_prompt}
> ```
>
> **Scene 2** - {location}
> ...
>
> 프롬프트 수정이 필요하신가요?

### 5. 최종 확인

> 챕터 {N} 대본이 완성되었습니다.
>
> - 대본: {char_count}자
> - 이미지 프롬프트: 3개
>
> 저장하고 다음 챕터로 진행할까요?
