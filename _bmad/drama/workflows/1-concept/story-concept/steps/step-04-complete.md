# Step 4: 완료

## 목표
스토리 컨셉 문서를 저장하고 다음 단계를 안내합니다.

## 수행 작업

### 1. 컨셉 문서 저장

생성된 스토리 컨셉을 프로젝트 폴더에 저장합니다:

```
{output_folder}/{project_name}/story-concept.md
{output_folder}/{project_name}/story-concept.yaml
```

### 2. 요약 출력

사용자에게 확정된 컨셉을 보기 좋게 정리해서 보여줍니다:

```
┌────────────────────────────────────────┐
│ 📖 스토리 컨셉 확정                       │
├────────────────────────────────────────┤
│ 제목: {title}                           │
│ 테마: {theme}                           │
│                                        │
│ 로그라인:                               │
│ {logline}                              │
│                                        │
│ 주인공: {name} ({age}세)                 │
│ 핵심 갈등: {core_conflict}              │
│ 상징적 소품: {symbolic_object}           │
│                                        │
│ 저장 위치:                              │
│ {output_folder}/{project_name}/         │
└────────────────────────────────────────┘
```

### 3. 다음 단계 안내

> 스토리 컨셉이 완성되었습니다! 🎉
>
> 다음 단계로 **8챕터 청사진**을 생성할 수 있습니다.
>
> 명령어: `/drama:blueprint` 또는 메뉴에서 [BP] 선택
>
> 청사진에서는:
> - 8챕터 스토리 구조
> - 상세 등장인물 프로필
> - 챕터별 핵심 이벤트
>
> 를 작성합니다.
