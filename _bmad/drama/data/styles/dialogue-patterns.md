# 대사 패턴 가이드

## 대사의 역할
- 가족 간 미묘한 긴장과 갈등 표현
- 표면적 의미와 숨겨진 의미의 이중 구조
- 캐릭터의 성격과 관계 드러내기

## 서브텍스트 대사 (Subtextual Dialogue)

### 기본 패턴
표면적으로는 평범한 말이지만, 실제로는 다른 감정을 담고 있음

**예시 1: 시어머니와 며느리**
```
시어머니: "밥은 먹었니?"
(속마음: 왜 항상 내가 먼저 챙겨야 하지?)

며느리: "네, 간단히요."
(속마음: 어머님은 제가 뭘 하든 불만이시네요.)
```

**예시 2: 부부 사이**
```
남편: "오늘 뭐 했어?"
(속마음: 하루 종일 뭘 하는지 모르겠네.)

아내: "그냥... 이것저것요."
(속마음: 당신이 알아주기나 하나요?)
```

**예시 3: 모녀 사이**
```
어머니: "넌 참 좋겠다, 하고 싶은 거 다 하고."
(속마음: 나도 저렇게 살고 싶었는데...)

딸: "엄마도 하고 싶은 거 하시면 되잖아요."
(속마음: 왜 항상 나한테 죄책감을 주시는 거야?)
```

## 세대별 말투 특성

### 50-70대 여성 (주인공)
- 존댓말과 반말의 적절한 사용
- 한탄조의 표현 ("아이고", "원...")
- 완곡한 거절 ("글쎄요", "생각해 볼게")

### 시어머니 (70-80대)
- 권위적이면서도 외로움이 묻어나는 말투
- 옛날 이야기 자주 언급
- 건강 관련 표현 자주 사용

### 며느리 (30-40대)
- 공손하지만 긴장이 느껴지는 말투
- 핵심을 피해가는 답변
- 시댁과 친정 사이 줄타기

### 남편 (50-60대)
- 무심한 듯한 짧은 대답
- 갈등 회피적 표현
- 가부장적 어조 (변화 과정에서 부드러워짐)

### 자녀 (20-30대)
- 현대적이고 직설적인 표현
- 부모 세대와의 가치관 충돌
- 진심을 표현하기 어려워함

## 갈등 대사 패턴

### 1단계: 불편함 암시
```
"...그래요."
"...네."
(짧고 건조한 대답으로 불편함 표현)
```

### 2단계: 간접적 공격
```
"어머님은 원래 그러셨잖아요."
"엄마는 항상 그런 식이야."
```

### 3단계: 직접적 충돌
```
"정말 더 이상은 못 참겠어요!"
"내가 뭘 그렇게 잘못했는데요!"
```

### 4단계: 진심 폭발
```
"알아요?! 내가 얼마나..."
(울음과 함께 30년 쌓인 감정 터뜨림)
```

## 화해 대사 패턴

### 사과와 용서
```
"미안해... 그때 내가 너무했어."
"아니에요... 저도요."
```

### 이해와 공감
```
"네 마음을 이제야 알겠어."
"어머니도... 힘드셨구나."
```

### 새로운 시작
```
"앞으로는 달라질 거야."
"우리, 다시 시작해보자."
```

## 대사 블록 형식
```
[인물명]
"대사 내용"

(괄호 안에 속마음 또는 행동 묘사)
```
