# 감정 표현 가이드

## 기본 원칙
- "설명하지 말고 느끼게 하라" (Feel, Don't Explain)
- 직접적 감정 명명보다 행동과 반응으로 표현
- 50-70대 여성의 감정 표현 방식 반영

---

## 슬픔 / 서러움

### 억눌린 슬픔
- 설거지하다 멈추고 창밖을 바라봄
- 빨래 널다가 한숨
- 손주 옷 개키다 눈물 훔침
- 혼자 있을 때 조용히 우는 모습

### 폭발하는 슬픔
- 30년 쌓인 눈물이 터지는 순간
- 말보다 울음이 앞서는 상황
- 주저앉아 흐느끼는 장면

### 관련 표현
```
"가슴이 먹먹해지는..."
"목이 메어오는..."
"참아왔던 눈물이..."
"속이 새까맣게 타들어가는..."
```

---

## 분노 / 억울함

### 참고 있는 분노
- 입술을 꾹 깨무는 모습
- 손에 힘이 들어가는 순간
- 고개를 돌려 시선 회피
- 떨리는 목소리로 대답

### 폭발하는 분노
- 쌓였던 말들이 쏟아지는 순간
- 목소리가 높아지며 떨림
- 평소 하지 않던 말이 튀어나옴

### 관련 표현
```
"속에서 뭔가 끓어오르는..."
"더 이상 참을 수 없어서..."
"수십 년 삼켜왔던 말들이..."
"가슴이 불덩이처럼..."
```

---

## 외로움 / 고독

### 일상 속 외로움
- 텅 빈 집에서 TV 소리만 울리는 모습
- 식탁에 혼자 앉아 밥 먹는 장면
- 전화기를 바라보지만 걸지 못하는 순간
- 옛 사진을 꺼내보는 밤

### 관계 속 외로움
- 가족들 사이에서도 느끼는 소외감
- 말해도 이해받지 못하는 답답함
- "나만 이상한 건가?" 하는 의문

### 관련 표현
```
"사람들 사이에서도 혼자인 것 같은..."
"아무도 내 마음을 모르는..."
"이 집에서 나만 투명인간 같은..."
"손 뻗으면 닿을 거리인데 너무 먼..."
```

---

## 후회 / 자책

### 과거에 대한 후회
- 그때 그렇게 하지 말 걸
- 한 마디만 더 했더라면
- 왜 그때 참았을까

### 자책
- 내가 부족해서 그랬나
- 더 잘할 수 있었는데
- 엄마로서, 아내로서 부족했던 것 같아

### 관련 표현
```
"가슴 한구석이 서늘해지는..."
"그때로 돌아갈 수 있다면..."
"평생 짊어질 짐 같은..."
"잊으려 해도 자꾸 떠오르는..."
```

---

## 기쁨 / 감동

### 잔잔한 기쁨
- 손주가 "할머니" 부르는 순간
- 오랜만에 온 가족이 모인 식탁
- 자녀가 성공했다는 소식

### 벅찬 감동
- 드디어 인정받는 순간
- 숨겨왔던 사랑을 확인할 때
- 화해의 포옹

### 관련 표현
```
"눈가가 뜨거워지는..."
"가슴이 벅차오르는..."
"입가에 미소가 번지는..."
"온몸에 따뜻한 기운이..."
```

---

## 희망 / 용기

### 새로운 시작
- 아직 늦지 않았다는 깨달음
- 나를 위한 삶도 괜찮다는 수용
- 다시 시작해볼 용기

### 관련 표현
```
"어둠 끝에 빛이 보이는 것 같은..."
"오랜만에 가슴이 두근거리는..."
"아직 할 수 있다는..."
"새로운 아침이 밝는 것 같은..."
```

---

## 복합 감정 (엔딩용)

### 슬픔 + 기쁨
> 기쁜데 왜 눈물이 나는지... 웃으면서 우는 이상한 내 모습이.

### 후회 + 희망
> 늦었지만, 아직 늦지 않았다고... 그렇게 믿고 싶습니다.

### 서러움 + 해방
> 드디어 끝났다는 안도와, 그래도 지나온 시간이 아까운...

### 분노 + 용서
> 미워했는데... 그래도... 가족이니까요.
