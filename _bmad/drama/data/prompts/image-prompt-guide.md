# 이미지 프롬프트 생성 가이드

## 목적
각 챕터의 핵심 감정 장면을 시각적으로 표현하기 위한 이미지 프롬프트 생성

## 프롬프트 구성 요소

### 1. 상황 설명 (Situation)
- 장면의 배경과 상황을 2-3줄로 요약
- 시간, 장소, 분위기 포함

### 2. 등장인물 (Characters)
- 이름, 연령대, 관계 명시
- 예: "김미숙, 60대, 주인공 / 박민주, 30대, 며느리"

### 3. 시선 처리 (Gaze)
**중요: 카메라를 직접 쳐다보지 않도록 지정**
- 각 인물이 바라보는 구체적인 대상 명시
- 예: "창밖", "상대방의 눈", "테이블 위 사진"

### 4. 장면 묘사 (Scene Description)

#### 자세와 행동 (Posture and Action)
- 서있는지, 앉아있는지
- 구체적인 동작 (팔 위치, 손 동작 등)
- 예: "테이블에 팔꿈치를 올리고 앞으로 기대어"

#### 표정 (Expression)
감정별 구체적 표현:

**눈물/슬픔:**
- 눈가에 살짝 맺힌 눈물
- 떨어지는 눈물방울
- 붉어진 눈두덩
- 빨개진 코끝

**분노:**
- 꽉 다문 입술
- 치켜 올라간 눈썹
- 긴장된 턱선
- 주먹 쥔 손
- 떨리는 목소리의 흔적

**기쁨:**
- 눈가의 주름
- 환한 미소
- 반짝이는 눈
- 자연스럽게 올라간 입꼬리
- 밝아진 안색

**슬픔:**
- 고개 숙인 자세
- 처진 어깨
- 멍한 시선
- 굳게 다문 입
- 창밖을 바라보는 시선

#### 배경과 소품 (Background and Objects)
- 시간적, 공간적 배경
- 분위기를 나타내는 주요 소품
- 예: "저녁 카페, 테이블 위 김이 나는 커피 두 잔, 창 너머 희미하게 보이는 도시 야경"

#### 상호작용 (Interaction)
- 인물 간 관계나 사건 흐름을 보여주는 상호작용
- 예: "A의 몸이 B에게 뭔가 말하려는 듯 살짝 기울어져"

## 제외할 요소 (MUST EXCLUDE)
- 아트 스타일 지정
- 아티스트 이름
- 조명 설정
- 카메라 앵글
- 렌즈 타입
- 색감/톤 지정

## 출력 형식

```
[Chapter X - Scene Y]
*Location:* [챕터 스크립트 내 위치 (예: 챕터 도입부, A가 진실을 고백하는 부분)]
*Situation:* [장면 배경 및 전체 상황 2-3줄 요약]
*Characters:*
  * [Character A: 이름, 연령대, 관계]
  * [Character B: 이름, 연령대, 관계]
*Scene Description:*
  * *Character A:*
    * *Posture and Action:* [구체적 자세와 동작]
    * *Expression:* [감정과 강도를 구체적으로]
    * *Gaze:* [시선이 향하는 대상]
  * *Character B:*
    * *Posture and Action:* [구체적 자세와 동작]
    * *Expression:* [감정과 강도를 구체적으로]
    * *Gaze:* [시선이 향하는 대상]
  * *Background and Objects:* [시공간적 배경과 주요 소품]
  * *Interaction:* [인물 간 상호작용]
```

## 영문 변환 시 주의사항
- 한국 가정 환경 특유의 요소 유지 (밥상, 방석, 한옥 요소 등)
- 한국인 외모 특성 반영
- 한국식 감정 표현 뉘앙스 유지
