# 대본 생성 프롬프트 가이드

## 입력 변수
- `{chapter_number}`: 생성할 챕터 번호 (1-8)
- `{blueprint}`: 8챕터 청사진 데이터
- `{characters}`: 등장인물 정보 JSON
- `{previous_chapters}`: 이전 챕터 요약 (2챕터부터)

## 출력 형식

```json
{
  "chapters": {
    "chapter{N}": "대본 내용 (3000자 목표)"
  },
  "imagePrompts": {
    "scene1": {
      "location": "챕터 내 위치",
      "situation": "장면 상황 요약",
      "characters": "등장인물 정보",
      "imagePrompt": "영문 이미지 프롬프트"
    },
    "scene2": { ... },
    "scene3": { ... }
  }
}
```

## 3가지 서사 도구

### 1. 내레이션 (일상/감정 묘사)
- 주인공의 목소리로 가족의 일상, 분위기, 상황 변화 묘사
- 여성의 섬세한 시선으로 포착
- 격식체 종결어미 사용 (-습니다, -ㅂ니다)

**예시:**
> 설거지를 하며 내쉬는 한숨, 손주 옷을 개키며 느끼는 뿌듯함,
> 시어머니 방에서 풍기는 약 냄새까지...

### 2. 대사 (가족 간 미묘한 긴장)
- 표면적으로는 평범하지만 속으로는 다른 생각을 품은 대화
- 독립된 '대사 블록' 형식 사용
- 가족 관계의 미묘한 긴장감 표현

**예시:**
```
"밥 먹었어?"
(속으로: 왜 항상 내가 먼저 물어야 하지?)
```

### 3. 내적 독백 (여성의 솔직한 속마음)
- 주인공의 내면 대화
- 가족에 대한 진심, 원망, 걱정 표출
- 여성 특유의 섬세하면서도 강렬한 감정 표현

## 후킹 전략

### 오프닝 훅 (처음 30초)
- 가장 충격적인 '갈등 장면' 또는 '눈물 대화' 회상으로 시작
- "이게 우리 가족 이야기인가?" 느낌 유도

### 마이크로 훅 (매 순간)
**감정 샌드위치 구조** 반복:
1. 일상 상황 제시 (내레이션)
2. 가족 갈등 (대사)
3. 감정적 상처 (내적 독백)

### 본질 훅 (챕터 끝)
- 가족 비밀이나 관계 변화에 대한 궁금증 유발
- "그때 무슨 일이 있었던 거지?"
- "이 비밀이 밝혀지면 우리 가족은 어떻게 될까?"

## 문장 리듬 조절
- 모든 문장을 같은 격식체로 끝내지 않기
- 연결 어미 활용 (-고, -며, -니까 등)
- 가족에게 이야기하듯 친근한 리듬 만들기

## 장면 완급 조절
- 격렬한 가족 갈등 장면 후에는
- 주인공이 혼자 차를 마시며 창밖을 바라보거나
- 손주 사진을 보며 스스로를 위로하는 잔잔한 장면 배치
