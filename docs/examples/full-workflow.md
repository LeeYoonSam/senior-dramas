# 예시 1: 전체 워크플로우

처음부터 끝까지 전체 워크플로우를 실행하는 예시입니다.

## 시나리오

"시어머니와 며느리의 30년 갈등 끝에 화해하는 이야기"를 만들어봅니다.

---

```
      따뜻한 가족 드라마를 함께 만들어볼까요?

      무엇을 도와드릴까요?

      [1] 새 스토리 시작하기

      [3] 대본 생성하기
      [4] 이미지 프롬프트 생성하기

      [6] 프롬프트 테스트하기
```

## Step 2: 스토리 컨셉 생성

```
미영: 아, 정말 공감 가는 주제네요!
      많은 분들이 눈물 흘리실 것 같아요.

      몇 가지 여쭤볼게요.

      1. 갈등의 시작은 어떤 사건이었나요?
         (예: 결혼 반대, 육아관 차이, 오해 등)
```

```
사용자: 결혼 초기에 며느리가 시어머니 몰래
       신혼여행 가는 걸 남편이 숨긴 게 발각되면서요
```

```
미영: 아, 사소하지만 깊은 상처가 되었군요.
      '나를 무시한다'는 오해가 30년을 가버린 거네요.

      2. 두 분 사이를 이어주는 손녀는 어떤 아이인가요?
```

```
사용자: 12살 손녀인데, 할머니랑 엄마 사이가
       안 좋은 걸 알고 있어요.
       학교에서 가족 과제를 하면서 화해를 시도해요.
```

```
미영: 손녀의 순수한 마음이 다리가 되어주는 거군요!
      정말 따뜻한 설정이에요.

      스토리 컨셉을 정리해볼게요...

      ━━━━━━━━━━━━━━━━━━━━━━━
      📖 스토리 컨셉
      ━━━━━━━━━━━━━━━━━━━━━━━

      제목: 시어머니의 마음

      원라인: 30년 갈등 끝에 손녀의 가족 과제로

      테마: 화해 (reconciliation)

      주요 캐릭터:
      • 김순자 (68세) - 시어머니

      • 이준혁 (45세) - 아들/남편
      • 이하늘 (12세) - 손녀

      핵심 갈등:
      결혼 초기 신혼여행 건으로 시작된 오해

      이대로 진행할까요?

      [2] 수정하기
```

## Step 3: 챕터 블루프린트 생성

```
미영: 8장 구조의 블루프린트를 설계할게요...

      [진행 중...]

      ━━━━━━━━━━━━━━━━━━━━━━━
      📋 챕터 블루프린트
      ━━━━━━━━━━━━━━━━━━━━━━━

      1장: 얼어붙은 가족
      • 명절, 서먹한 분위기
      • 하늘이가 할머니와 엄마 사이 갈등 목격

      2장: 가족 과제
      • 하늘이 학교 과제: "우리 가족 이야기"
      • 할머니 인터뷰 계획

      3장: 첫 번째 대화
      • 하늘이가 할머니에게 인터뷰 요청
      • 순자, 30년 전 이야기 시작

      4장: 엄마의 이야기
      • 하늘이가 엄마에게도 인터뷰
      • 민지의 30년 묵은 서운함

      5장: 진실
      • 아빠에게 들은 진짜 이야기
      • 모두가 서로를 위했던 마음

      6장: 후회
      • 순자, 민지 각각 과거 회상
      • 깨달음의 눈물

      7장: 화해
      • 하늘이의 과제 발표 날
      • 두 사람의 진심 어린 대화

      8장: 새로운 가족
      • 3세대가 함께하는 따뜻한 일상
      • 새로운 명절

      ━━━━━━━━━━━━━━━━━━━━━━━

      다음 단계로 넘어갈까요?

      [1] 대본 생성하기
      [2] 블루프린트 수정하기
```

## 소요 시간

| 단계 | 예상 시간 |
|------|----------|
| 스토리 컨셉 | 5-10분 |
| 블루프린트 | 3-5분 |
| 대본 (8장) | 20-30분 |
| 이미지 프롬프트 | 5-10분 |
| 내보내기 | 1-2분 |
| **총합** | **35-60분** |

---

## 팁

1. **챕터별 생성 권장**: 전체 생성보다 챕터별 확인이 품질 좋음
2. **중간 저장**: 각 단계마다 자동 저장됨
3. **수정 용이**: 언제든 이전 단계로 돌아가 수정 가능
