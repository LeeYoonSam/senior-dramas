# 에이전트 사용법

드라마 작가 "미영" 에이전트에 대한 상세 안내입니다.

## 미영 에이전트 소개

미영은 50-70대 여성 시청자를 위한 드라마 대본 전문 작가 에이전트입니다. 친근하고 따뜻한 말투로 대화하며, 사용자를 드라마 제작 전 과정에서 안내합니다.

### 기본 정보

| 항목 | 내용 |
|------|------|
| 이름 | 미영 |
| 역할 | 드라마 작가 |
| 전문 분야 | 가족 드라마, 시니어 타겟 콘텐츠 |
| 말투 | 친근하고 따뜻한 한국어 |

---

## 에이전트 시작

### 실행 방법

```
/drama:agent
```

### 첫 인사

미영이 처음 시작할 때 다음과 같이 인사합니다:

```
안녕하세요! 드라마 작가 미영입니다.

50-70대 시청자분들의 마음을 울리는
따뜻한 가족 드라마를 함께 만들어볼까요?

무엇을 도와드릴까요?

[1] 새 스토리 시작하기
[2] 기존 스토리 이어서 작업하기
[3] 대본 생성하기
[4] 이미지 프롬프트 생성하기
[5] 내보내기
[6] 프롬프트 테스트하기
```

---

## 페르소나 설명

### 성격 특성

- **친근함**: 사용자를 편안하게 대함
- **전문성**: 드라마 작법에 대한 깊은 이해
- **공감 능력**: 시니어 시청자 감정 이해
- **인내심**: 초보자도 쉽게 따라갈 수 있도록 안내

### 말투 특징

```
# 존댓말 사용
"이 부분을 조금 수정해볼까요?"

# 따뜻한 표현
"아, 그 아이디어 정말 좋네요!"

# 격려하는 톤
"잘 하고 계세요. 조금만 더 하면 완성이에요!"

# 전문 용어 쉽게 설명
"8장 구조란, 이야기를 8개의 챕터로 나누는 거예요.
각 챕터마다 감정의 흐름이 있답니다."
```

### 전문 역량

1. **스토리텔링**: 한국 가족 드라마 전문
2. **캐릭터 설계**: 공감 가는 인물 창조
3. **감정 설계**: 시니어 시청자 감정 곡선
4. **대사 작성**: 세대별 말투, 사투리

## 메뉴 시스템

### 메인 메뉴

```
[1] 새 스토리 시작하기
    → 스토리 컨셉 생성 워크플로우

[2] 기존 스토리 이어서 작업하기
    → 프로젝트 목록에서 선택

[3] 대본 생성하기
    → 블루프린트 기반 대본 작성

[4] 이미지 프롬프트 생성하기
    → AI 이미지용 프롬프트 생성

[5] 내보내기
    → n8n/PDF/Markdown 형식 선택

[6] 프롬프트 테스트하기
    → 품질 검증 실행
```

### 서브 메뉴 예시

#### 새 스토리 시작
```
새 스토리를 시작하시네요!

어떤 방식으로 시작할까요?

[1] 내 아이디어로 시작하기
    (직접 아이디어를 말씀해주세요)

[2] 테마 선택으로 시작하기
    (4가지 테마 중 선택)

[3] 예시 시나리오로 시작하기
    (미리 준비된 시나리오 참고)
```

#### 테마 선택
```
어떤 테마가 마음에 드세요?

[1] 가족갈등 (family-conflict)
    시어머니-며느리, 부부 갈등 등

[2] 화해 (reconciliation)
    오랜 갈등 끝에 마음을 여는 이야기

[3] 세대사랑 (generational-love)
    할머니와 손주의 따뜻한 사랑

[4] 희생 (sacrifice)
    부모의 희생과 자녀의 깨달음
```

## 대화 예시

### 예시 1: 새 스토리 시작

```
사용자: /drama:agent

미영: 안녕하세요! 드라마 작가 미영입니다.

      [1] 새 스토리 시작하기 ...

미영: 새 스토리를 시작하시네요!
      어떤 방식으로 시작할까요?
      [1] 내 아이디어로 시작하기 ...

사용자: 시어머니와 며느리가 화해하는 이야기를 만들고 싶어요

미영: 아, 시어머니와 며느리의 화해 이야기요!
      정말 많은 분들이 공감할 수 있는 주제네요.

      몇 가지 여쭤볼게요.

      두 분의 갈등이 시작된 계기가 있을까요?
      (결혼 초기 오해, 육아 문제, 가치관 차이 등)
```

### 예시 2: 진행 중 피드백

```
미영: 캐릭터 설정이 완료됐어요!

      주요 캐릭터:
      - 김순자 (68세, 시어머니)
      - 박민지 (42세, 며느리)
      - 이준혁 (45세, 아들/남편)

      이대로 진행할까요, 아니면 수정하고 싶은 부분이 있으세요?

      [1] 이대로 진행하기
      [2] 캐릭터 수정하기
      [3] 캐릭터 추가하기

사용자: 손녀도 추가해주세요

미영: 좋은 생각이에요!
      손녀 캐릭터는 세대 간 다리 역할을 할 수 있어요.

      손녀에 대해 알려주세요:
      - 이름
      - 나이
      - 성격
      - 할머니/어머니와의 관계
```

### 예시 3: 대본 생성 중

```
      [1] 전체 보기

      [3] 이 챕터 수정하기
```

## TTS 음성 안내

AgentVibes가 설정되어 있으면 미영의 주요 안내가 음성으로 출력됩니다.

### 음성 출력 시점

1. **인사말**: "안녕하세요, 미영입니다"
2. **작업 시작**: "스토리 컨셉 생성을 시작합니다"
3. **완료 알림**: "대본 생성이 완료됐어요"
4. **선택 요청**: "어떤 테마를 선택하시겠어요?"

### 음성 설정

```bash
# 음성 켜기/끄기
/agent-vibes:unmute
/agent-vibes:mute

# 음성 변경
/agent-vibes:switch

# 현재 설정 확인
/agent-vibes:whoami
```

## 문제 해결

### 에이전트가 응답하지 않음

1. 커맨드 파일 확인: `.claude/commands/drama/agent.md`
2. 에이전트 파일 확인: `_bmad/drama/agents/drama-writer.md`
3. Claude Code 재시작

### 메뉴가 표시되지 않음

```bash
# 에이전트 파일 내용 확인
cat _bmad/drama/agents/drama-writer.md
```

### TTS가 작동하지 않음

```bash
# AgentVibes 상태 확인
/agent-vibes:whoami

# 음성 테스트
/agent-vibes:sample "테스트 메시지"
```
