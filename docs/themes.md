# 테마 가이드

Senior Dramas 시스템에서 사용할 수 있는 4가지 드라마 테마에 대한 상세 가이드입니다.

## 테마 개요

| 테마 | 한글명 | 핵심 감정 | 주요 관계 |
|------|--------|----------|----------|
| family-conflict | 가족갈등 | 분노, 서운함 | 시월드, 부부 |
| reconciliation | 화해 | 후회, 용서 | 부모-자녀, 형제 |
| generational-love | 세대사랑 | 그리움, 이해 | 조부모-손주 |
| sacrifice | 희생 | 감사, 미안함 | 부모-자녀 |

---

## family-conflict (가족갈등)

### 설명
가족 내 갈등과 오해를 다루는 테마입니다. 시어머니-며느리, 부부 갈등 등 현실적인 가족 문제를 조명합니다.

### 핵심 감정 키워드
- 분노
- 서운함
- 억울함
- 답답함
- 외로움

### 갈등 유형
- **시월드 갈등**: 시어머니-며느리, 시누이-올케
- **부부 갈등**: 소통 부재, 가치관 차이
- **고부 갈등**: 육아관, 생활방식 충돌
- **재산 갈등**: 상속, 부양 문제

### 관계 역학
```
시어머니 ←──갈등──→ 며느리
    │                    │
    └───────아들────────┘
           (중재자)
```

### 대표 시나리오
1. "30년 묵은 오해" - 사소한 오해가 큰 갈등으로
2. "명절 전쟁" - 매년 반복되는 명절 갈등
3. "손주 육아 전쟁" - 육아관 차이로 인한 갈등

### 사용 예시
```yaml
theme: family-conflict

main_conflict:
  type: "시월드 갈등"
  trigger: "결혼 초기 오해"
  duration: "20년"

emotional_arc:
  start: "억눌린 분노"
  middle: "폭발과 대립"
  end: "이해와 화해"
```

### 권장 감정 곡선
```
1장: 긴장 ─────────┐
2장: 분노 ─────────┤
3장: 희망 ───┐     │
4장: 시도 ───┼─────┤
5장: 절망 ───┘     │
6장: 깨달음 ───────┤
7장: 화해 ─────────┤
8장: 평화 ─────────┘
```

## reconciliation (화해)

### 설명
오랜 갈등 끝에 진심을 마주하고 화해하는 과정을 다룹니다. 용서와 이해의 여정을 그립니다.

### 핵심 감정 키워드
- 후회
- 용서
- 미안함
- 그리움
- 따뜻함

### 갈등 유형
- **부모-자녀**: 진로 갈등, 결혼 반대
- **형제자매**: 어린 시절 상처, 편애
- **친구**: 오해로 인한 절교
- **부부**: 위기 후 재결합

### 관계 역학
```
과거의 나 ←──후회──→ 현재의 나
    │                     │
    └────── 상대방 ───────┘
           (화해 대상)
```

### 대표 시나리오
1. "10년 만의 전화" - 연락 끊긴 딸과의 재회
2. "마지막 선물" - 투병 중 화해하는 형제
3. "아버지의 일기" - 돌아가신 후 발견한 진심

### 사용 예시
```yaml
theme: reconciliation

main_conflict:
  type: "부모-자녀 갈등"
  cause: "결혼 반대로 인한 단절"
  separation: "10년"

emotional_arc:
  start: "원망과 미움"
  turning_point: "위기 상황"
  end: "진심 전달과 용서"
```

### 권장 감정 곡선
```
1장: 냉담 ─────────┐
2장: 회상 ─────────┤
3장: 그리움 ──┐    │
4장: 접근 ────┼────┤
5장: 거절 ────┘    │
6장: 위기(병/사고) ┤
7장: 화해 ─────────┤
8장: 새 시작 ──────┘
```

## generational-love (세대사랑)

### 설명
조부모와 손주 사이의 따뜻한 사랑을 다룹니다. 세대 간 이해와 소통의 아름다움을 보여줍니다.

### 핵심 감정 키워드
- 그리움
- 이해
- 존경
- 사랑
- 배움

### 갈등 유형 (가벼운)
- **세대 차이**: 디지털 vs 아날로그
- **가치관**: 현대 vs 전통
- **소통 방식**: 직접적 vs 우회적

### 관계 역학
```
할머니/할아버지
        │
    ────┼──── (세대 간 다리)
        │
      손주
```

### 대표 시나리오
1. "할머니의 스마트폰" - 손주가 가르치는 기술
2. "비밀 레시피" - 요리로 전해지는 사랑
3. "여름방학 일기" - 시골에서의 추억

### 사용 예시
```yaml
theme: generational-love

relationship:
  elder: "할머니 (78세)"
  younger: "손녀 (12세)"

emotional_arc:
  start: "서먹한 첫 만남"
  middle: "함께하는 시간들"
  end: "깊어진 유대"

key_moments:
  - "할머니의 옛날 이야기"
  - "손녀가 가르쳐주는 카톡"
  - "함께 만드는 송편"
```

### 권장 감정 곡선
```
1장: 서먹함 ──────────┐
2장: 호기심 ──────────┤
3장: 교감 ────────────┤
4장: 깊어짐 ──────────┤
5장: 이별 위기 ───────┤
6장: 그리움 ──────────┤
7장: 재회 ────────────┤
8장: 따뜻한 일상 ─────┘
```

## sacrifice (희생)

### 설명
부모의 희생과 자녀의 뒤늦은 깨달음을 다룹니다. 깊은 감동과 눈물을 자아내는 테마입니다.

### 핵심 감정 키워드
- 감사
- 미안함
- 깨달음
- 존경
- 눈물

### 갈등 유형
- **인식 갈등**: 당연함 vs 희생
- **세대 갈등**: 기대 vs 현실
- **자아 갈등**: 효도 vs 현실

### 관계 역학
```
부모 (희생)
    │
    ├── 과거: 당연하게 받음
    │
    └── 현재: 뒤늦은 깨달음
            │
          자녀
```

### 대표 시나리오
1. "엄마의 손" - 거친 손이 말해주는 사랑
2. "아버지의 도시락" - 새벽에 싸주던 도시락
3. "고등어 한 토막" - 반찬에 담긴 희생

### 사용 예시
```yaml
theme: sacrifice

sacrifice_type: "일상 속 희생"

parent:
  name: "어머니"
  sacrifice: "자신의 꿈을 포기하고 자녀 뒷바라지"

child:
  current_state: "성공한 직장인"
  realization_trigger: "어머니 입원"

emotional_arc:
  start: "무심한 일상"
  trigger: "어머니의 건강 이상"
  realization: "숨겨진 희생 발견"
  end: "감사와 효도 결심"
```

### 권장 감정 곡선
```
1장: 일상 ────────────┐
2장: 무심함 ──────────┤
3장: 작은 변화 ───────┤
4장: 불안 ────────────┤
5장: 위기 (입원 등) ──┤
6장: 발견과 깨달음 ───┤  ← 클라이맥스
7장: 후회와 결심 ─────┤
8장: 감사와 사랑 ─────┘
```

## 테마 선택 가이드

### 시청자 반응 예상

| 테마 | 예상 반응 | 강도 |
|------|----------|------|
| family-conflict | "맞아, 나도..." | 공감 ★★★★★ |
| reconciliation | "용서해야겠어..." | 감동 ★★★★☆ |
| generational-love | "우리 손주..." | 따뜻함 ★★★★★ |
| sacrifice | (눈물) | 눈물 ★★★★★ |

### 조합 권장

테마는 기본적으로 하나를 선택하되, 서브 테마로 다른 요소를 가미할 수 있습니다:

```yaml
main_theme: family-conflict
sub_theme: reconciliation

# 시어머니-며느리 갈등(main)이
# 결국 화해(sub)로 마무리되는 구조
```

### 테마별 권장 길이

| 테마 | 권장 챕터 수 | 이유 |
|------|-------------|------|
| family-conflict | 8장 | 갈등 심화에 시간 필요 |
| reconciliation | 8장 | 화해 과정 충분히 묘사 |
| generational-love | 6-8장 | 유연하게 조절 가능 |
| sacrifice | 8장 | 깨달음까지 충분한 빌드업 |
